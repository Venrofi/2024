App deployment:

  • To deploy this site use Google Cloud -> App Engine. It provides you with a server in the cloud. ☁️
  • For that you need Google Cloud account & Google Cloud command tools installed locally. 💻
  • With that done, add a `app.yaml` file and specify the runtime. Also add a proper start command in the `package.json`. 👨‍💻
  • The last step is to deploy by using command: `gcloud app deploy`! 🚀
